API使用说明

加密流程

1. 将JSON数据序列化为字符串

2. 使用Base64编码数据

3. 使用HMAC-SHA256生成签名

4. 发送包含加密数据和签名的POST请求

请求示例

// 请求体 { "data": "eyJuYW1lIjogIuWJjeaYjiIsICJpZCI6ICIxMjM0NTY3ODkwMTIzNDU2NzgiLCAidGltZXN0YW1wIjogMTY5MDAwMDAwMH0=", "signature": "a1b2c3d4e5f6...（64位十六进制字符串）" }

响应示例

// 成功响应 { "success": true, "data": "加密的响应数据", "signature": "响应数据签名", "message": "请求处理完成" } // 错误响应 { "success": false, "message": "错误描述" }
